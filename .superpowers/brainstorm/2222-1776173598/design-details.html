分层设计 + 插件包结构

分层架构设计

三层架构：编排层 → 适配层 → 插件层
┌────────────────────────────────────────────────────────────────────┐
│  第一层：编排层 (不变)                                               │
│  ────────────────────────────────────────────────────────────────  │
│  Orchestrator / ExecutionService                                    │
│    • 管理 Agent 生命周期（Spawn、Stop、状态监控）                    │
│    • 协调多 Agent 交互                                               │
│    • 不感知具体 Agent 类型                                           │
│                                                                     │
│  调用方式：adapterRegistry.GetAdapter(baseAgent.Type)           │
│  返回值：AgentAdapter 接口（统一）                                    │
├────────────────────────────────────────────────────────────────────┤
│  第二层：适配层 (新增)                                               │
│  ────────────────────────────────────────────────────────────────  │
│  AdapterRegistry                                                    │
│    • 维护 type → AdapterFactory 的映射                              │
│    • 提供 Register(type, factory, meta) 方法                        │
│    • 提供 GetAdapter(type) 方法                                     │
│    • 提供 GetTypes() 方法（返回所有已注册类型列表）                   │
│                                                                     │
│  AgentAdapter 接口（不变）                                           │
│    Execute、ExecuteWithStream、StartSession、ResumeSession...       │
├────────────────────────────────────────────────────────────────────┤
│  第三层：插件层 (重构)                                               │
│  ────────────────────────────────────────────────────────────────  │
│  每个基础 Agent 为独立插件包：                                       │
│                                                                     │
│  internal/service/agent/plugins/                                    │
│    ├── claude_code/                                                 │
│    │   ├── plugin.go      ←── Register() 入口                      │
│    │   ├── adapter.go     ←── ClaudeAdapter 实现                    │
│    │   └── config.go      ←── 默认配置（CLI路径、config目录等）      │
│    │                                                                │
│    ├── open_code/                                                   │
│    │   ├── plugin.go      ←── Register() 入口                      │
│    │   ├── adapter.go     ←── OpenCodeACPAdapter 实现（重命名）     │
│    │   ├── acp_adapter.go ←── ACP基础适配器（复用）                 │
│    │   └── config.go      ←── 默认配置                              │
│    │                                                                │
│    └── [内网Agent]/                                                  │
│        ├── plugin.go                                                │
│        ├── adapter.go                                               │
│        └── config.go                                                │
│                                                                     │
│  main.go 启动时：                                                    │
│    plugins.RegisterAll(adapterRegistry)                             │
│    或                                                                │
│    plugins.RegisterClaudeCode(adapterRegistry)                      │
│    plugins.RegisterOpenCode(adapterRegistry)                        │
│    // 内网单独编译时添加                                              │
│    plugins.RegisterInternalAgent(adapterRegistry)                   │
└────────────────────────────────────────────────────────────────────┘

单插件包结构详解

插件包目录结构
internal/service/agent/plugins/claude_code/
│
├── plugin.go              # 插件入口 - 注册到Registry
│   ┌─────────────────────────────────────────────────┐
│   │ func Register(r *agent.AdapterRegistry) {       │
│   │   r.Register(agent.PluginMeta{                  │
│   │     Type:        "claude_code",                 │
│   │     Name:        "ClaudeCode",  // 无空格        │
│   │     Description: "Anthropic Claude CLI",        │
│   │     CliPath:     "claude",      // 默认CLI路径   │
│   │     ConfigDir:   ".claude",     // 配置目录名    │
│   │     Factory:     NewClaudeAdapter,              │
│   │   })                                            │
│   │ }                                               │
│   └─────────────────────────────────────────────────┘
│
├── adapter.go             # ClaudeAdapter 实现
│   • 实现 AgentAdapter 接口
│   • 使用 stream-json 格式解析
│   • Session 管理
│
├── parser.go              # 输出解析器（可选，可复用）
│   • 解析 Claude 特有的 stream-json 格式
│
└── config.go              # 默认配置
│   • DefaultCliPath = "claude"
│   • DefaultConfigDir = ".claude"
│   • Environment vars mapping

AdapterRegistry 核心接口

// adapter_registry.go

type PluginMeta struct {
    Type        BaseAgentType      // "claude_code", "open_code"
    Name        string             // 显示名称 "ClaudeCode", "OpenCode"
    Description string             // 描述
    CliPath     string             // 默认CLI路径
    ConfigDir   string             // 配置目录名
    Factory     AdapterFactory     // func(baseAgent) AgentAdapter
}

type AdapterRegistry struct {
    plugins map[BaseAgentType]PluginMeta
}

func (r *AdapterRegistry) Register(meta PluginMeta)
func (r *AdapterRegistry) GetAdapter(baseAgent *model.BaseAgent) AgentAdapter
func (r *AdapterRegistry) GetTypes() []BaseAgentTypeInfo  // 替代原 GetTypes()
func (r *AdapterRegistry) GetMeta(type BaseAgentType) PluginMeta
